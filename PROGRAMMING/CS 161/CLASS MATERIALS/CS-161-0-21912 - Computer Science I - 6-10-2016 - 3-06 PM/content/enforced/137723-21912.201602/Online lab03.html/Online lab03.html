-- Lab03

This is the second lab for cs161. The objective of this lab is to
become familiar with a linux text editor. Specifically we will learn:

- about the two main linux text editors
- how to use (the basics) of one of these editors

-- Logging in

Use putty, as described in the introduction lesson, to login. Once
you are logged in change your current directory to the "labs"
directory (Hint: "cd labs").

-- Copy lab files

Each lab will have a number of lab files from my directory on the
server. To start each lab you will copy these files into your own
lab# directory. For lab03 type this command:

"cp ~mike.noel/onlinelabs/cs161/lab03/* ."

You must type this carefully. Pay close attention to the
punctuation. If you do this successfully then you will end up with
some new files in the lab03 directory. You can see this by listing
the files (Hint: "ls"). You may have noticed an error about omitting
a directory. Ignore that.

-- Reading the lab instructions

All of the labs include instructions in the lab files. This will
always be in a file called "readme". You can view the contents of
this file by typing "more readme". Use the space bar to move forward
pages. Type "q" to quit. From this point on all lab instructions
will be found in the readme for that lab.

-- No other instructions

The rest of the lab03 instructions are in the readme on the linux
server. In fact, for all future lessons it is not necessary to view
the lab file on D2L. You can just use the readme you find on the
linux server.
